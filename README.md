<h2># Strivers A2Z DSA Course/Sheet Solutions</h2>

!Strivers A2Z DSA Course

This repository contains my solutions to the problems from the **Strivers A2Z DSA Course/Sheet**.
## Course Overview

The Strivers A2Z DSA Solutions will cover essential topics, including:

1. **Basics**:
   - Data types
   - Conditional statements (if-else, switch)
   - Arrays, strings
   - Loops (for, while)
   - Functions
   - Time complexity analysis

2. **Sorting Techniques**:
   - Selection Sort
   - Bubble Sort
   - Insertion Sort
   - Merge Sort
   - Quick Sort

3. **Array Problems** (Easy to Hard):
   - Largest element
   - Second largest element
   - Check if sorted
   - Remove duplicates
   - Rotate arrays
   - Linear search
   - Union and intersection
   - Missing number
   - Maximum consecutive ones
   - ...

4. **Other Topics**:
   - Recursion
   - Hashing
   - Dynamic programming
   - ...
